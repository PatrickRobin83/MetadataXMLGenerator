Um den MetaDataXMLGenerator nutzen zu können bitte folgende Punkte ausführen:

1. Dateien entpacken
2. Settings.ini im Unterordner resources mit einem Text Editor öffnen
3. Web RootPath richtig eintellen
4. Lokalen RootPath richtig einstellen
5. MetaDataConsole.exe ausführen
6. Wenn die Konsole geöffnet ist kann entschieden werden ob Dateien erstellt, gelöscht werden sollen oder ob das Programm dirket geschlossen werden soll.
